Planned Improvements for ModeMaker

ModeMaker was released once it was free of known bugs, but it does have some limitations and features not yet implemeted:

  • Add the ability to make the canvas a circle rather than a rectangle
  • Move some currently-fixed parameters into a preferences window (e.g., the number of Monte Carlo points used for mode measurements)
  • Add new shapes: octagons (c.f., pyroxene), rhombi (c.f., amphiboles)
  • Add the ability to create compound shapes (e.g., multiple non-rotated rectangles that would mimic an igneous feldspar)
  • Redo the core functionality of the program as a Foundation Tool and put it on the web.
